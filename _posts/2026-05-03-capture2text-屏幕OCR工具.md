## 🎯 它是做什么的？

Capture2Text 是一个"屏幕文字提取器"。它不是截图工具，而是一个**OCR（光学字符识别）工具**，可以：

- ✅ **快速提取屏幕文字**：用快捷键框选屏幕上的文字，自动识别并转为可编辑文本
- ✅ **支持 100+ 种语言**：英语、中文（简/繁）、日语、韩语等
- ✅ **多种捕获模式**：标准框选、文本行、气泡文本等
- ✅ **便携式**：无需安装，解压即用
- ✅ **免费开源**：GNU GPL 许可，完全免费
- ✅ **仅支持 Windows**：Windows 7/8/10

**真实场景**：你在 PDF 或视频截图上看到一段文字想复制，但无法选中文字
- 传统方法：手动打字 😫
- Capture2Text：快捷键框选 → 文字自动到剪贴板 ⚡

## 🔑 核心概念解读

**OCR（光学字符识别）**
- 专业说：将图像中的文字转换为可编辑的计算机文本
- 通俗说：让电脑"看懂"图片里的字，像你拍照识别文字一样

**便携式软件**
- 专业说：无需安装程序，可直接运行的独立可执行文件
- 通俗说：下载解压就能用，不用安装也不写注册表，放 U 盘随处携带

**气泡 OCR**
- 专业说：自动识别并提取漫画对话气泡内的文本
- 通俗说：专门为看漫画设计，鼠标点一下对话框，文字自动出来

**Furigana（振假名）**
- 专业说：日语注音辅助字符
- 通俗说：日语汉字旁边标注的小假名，Capture2Text 可以自动去除

## 🚀 主要功能特性

### 捕获模式
1. **标准 OCR 捕获**：框选任意区域文字
2. **文本行捕获**：自动识别鼠标最近的一行文字（`Windows Key + E`）
3. **前向文本行捕获**：从鼠标位置开始向前捕获（`Windows Key + W`）
4. **气泡 OCR**：自动识别漫画对话框（`Windows Key + S`）

### 高级功能
- **多语言支持**：100+ 种语言
- **自动翻译**：识别后自动翻译成其他语言
- **文本转语音**：识别后朗读出来
- **命令行支持**：可批量处理图片文件
- **快捷键切换**：快速切换 3 种常用语言（`Windows Key + 1/2/3`）

### 文本方向支持（中文/日语）
- **横向文字**：从左到右
- **纵向文字**：从上到下
- **自动检测**：根据框选区域自动判断（`Windows Key + O`）

## 🚀 快速开始

### 下载和使用
1. 从 SourceForge 下载最新版本
2. 解压 zip 文件
3. 双击 `Capture2Text.exe` 启动
4. 右下角托盘区会出现图标

### 基础操作
```bash
# 标准框选识别
Windows Key + Q  # 开始框选
再次按下或 Enter  # 完成识别

# 文本行快速识别
Windows Key + E  # 自动识别鼠标最近的一行

# 气泡识别（漫画）
Windows Key + S  # 识别对话框内容

# 切换语言（快捷键）
Windows Key + 1/2/3  # 切换预设的 3 种语言
```

### 命令行使用
```bash
# OCR 图片文件
Capture2Text_CLI.exe -i image.png -o result.txt

# 批量处理
Capture2Text_CLI.exe -i img1.png -i img2.jpg -o result.txt

# 指定语言
Capture2Text_CLI.exe -l "Chinese - Simplified" -i image.png
```

## 🎯 实用技巧

### 提高识别准确率
1. **选择正确语言**：在设置中选择要识别的语言
2. **调整框选区域**：尽量只框选文字，避免多余空白
3. **使用文本行模式**：单行文字用 `Windows Key + E` 更准确

### 日语学习最佳实践
1. 设置 OCR 语言为日语
2. 使用气泡 OCR 提取对话（`Windows Key + S`）
3. 自动去除 Furigana（振假名）
4. 配合 JGlossator 查词工具使用

### 多语言快速切换
- 在设置中配置 3 个常用语言
- 使用 `Windows Key + 1/2/3` 快速切换
- 适合需要识别多种语言的场景
