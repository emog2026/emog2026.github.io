### 1. 数据盘初始化挂挂载
```
yum install -y parted
parted /dev/sdb # 分区
mklabel gpt
mkpart primary 1MiB 100% 
align-check optimal 1
print
quit
partprobe
yum install -y e2fsprogs
mkfs -t ext4 /dev/sdb1 # 创建文件系统
mount /dev/sdb1 /nas/test
echo "/dev/sdb1 /nas/test ext4 defaults 0 0" >> /etc/fstab
```
