# U钙网"春花秋月"Logo设置指南

## 🎯 目标
将U钙网(uugai.com)制作的"春花秋月"logo设置为博客头像

## 📝 操作步骤

### 1. 制作/获取Logo
1. 访问：https://www.uugai.com
2. 在首页输入框输入：春花秋月
3. 点击"生成LOGO"或"开始设计"
4. 浏览不同样式，选择满意的设计
5. 点击下载，保存为PNG格式（推荐，背景透明）

### 2. 文件命名
将下载的logo重命名为：`chunhuaqiuyue-logo.png`

### 3. 放置位置
将文件放到：`img/chunhuaqiuyue-logo.png`

### 4. 更新配置
在`_config.yml`中更新：
```yaml
sidebar-avatar: /img/chunhuaqiuyue-logo.png
```

## 🎨 推荐样式选择
- **字体风格**：中文书法字体或现代简约字体
- **颜色搭配**：
  - 黑色文字 + 透明背景（推荐）
  - 深蓝色文字（科技感）
  - 绿色点缀（呼应现有主题）
- **图标元素**：
  - 花朵图案（呼应"春花"）
  - 月亮图案（呼应"秋月"）
  - 四季元素

## 📐 技术要求
- **格式**：PNG（支持透明背景）
- **尺寸**：建议500x500px或更大
- **文件大小**：尽量<500KB
- **背景**：透明或白色

## 🔧 完成后检查
1. 访问 http://127.0.0.1:4000/
2. 查看侧边栏头像是否正确显示
3. 检查About Me页面头像
4. 确认在不同屏幕尺寸下的显示效果
